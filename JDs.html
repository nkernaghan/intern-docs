About Haven256

Haven256 is a crypto forensics and digital asset intelligence firm. We build tools that trace illicit funds, detect scam wallets, and support law enforcement and compliance teams globally.

The Role

You'll own a real internal project from day one — building AI and blockchain intelligence tools that solve operational problems. No busywork. Your code ships. By week 12 you'll hand off a working prototype to the team.

Projects Available
Crypto Mixer Tracer
Internal AI OS
Wallet Drift Monitor
Estate Planning Engine
What You'll Build
  • Python backends and REST APIs using FastAPI
  • AI/LLM integrations — Claude API, RAG pipelines, vector databases
  • Blockchain data ingestion via Helius, Alchemy, and Blockchair APIs
  • Frontend dashboards using Next.js / React
  • Deployed production builds on Railway or Vercel
Requirements
  • Strong Python — required
  • Comfort with APIs and relational databases (PostgreSQL, Supabase)
  • Exposure to LLMs, embeddings, or vector search — preferred
  • Basic blockchain / crypto knowledge — nice to have
  • Ability to work independently and ship real code
Skill Levels

We're hiring across experience levels — junior, mid, and senior tracks. You'll be scoped and evaluated against your own track, not against others.

Deliverables by Week 12
  • Working prototype with core features operational
  • GitHub repository with full commit history
  • Live demo to Haven256 team
  • Handoff bundle: README, architecture doc, deployment guide
