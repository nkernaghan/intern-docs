Haven256 AI & Cybersecurity Courses
About This Guide

The courses below are all free, self-paced, and publicly accessible — no corporate login or paid subscription required. Each one can be completed in a couple hours or less.

AI Fundamentals — Anthropic Skilljar
01
AI Capabilities and Limitations
An introductory course about how AI works — what it can do, where it falls short, and how to think about its role in real work.
Why This First Grounds your expectations before you start using AI tools daily. Especially important for forensics and compliance work where hallucinations carry real risk.
02
AI Fluency: Framework & Foundations
Learn to collaborate with AI systems effectively, efficiently, ethically, and safely. Covers the mental models for working alongside AI as a tool.
Why This Course Builds the "how to think about AI" layer that our intern curriculum assumes but doesn't teach directly. Sets up the right habits for the entire 12 weeks.
03
Claude 101
Learn how to use Claude for everyday work tasks, understand core features, and explore resources for more advanced learning.
Why This Course You'll be using Claude constantly during the program — for research, drafting, code review, and analysis. Get proficient before you walk in the door.
04
Building with the Claude API
Comprehensive course covering the full spectrum of working with Anthropic models using the Claude API — authentication, prompting, streaming, tool use, and more.
Why This Course All four intern projects involve building AI-powered tools. This gives you the hands-on API knowledge to start writing real code from Week 2.
05
Introduction to Subagents
Learn how to use and create sub-agents in Claude Code to manage context, delegate tasks, and build specialized workflows that keep your main conversation clean and focused.
Why This Course Relevant for the multi-module architectures in Projects 2 and 3. Understanding agent delegation will help you build systems where components work semi-autonomously.
